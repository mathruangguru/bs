1.
Diketahui fungsi $f(x) = 2x + 3$ dan $g(x) = x - 5$. Hasil dari operasi aljabar fungsi $(f + g)(x)$ adalah ...
  1. $3x - 2$
  2. $3x + 8$
  3. $2x - 2$
  4. $3x - 8$
  5. $x + 8$
2.
Jika $f(x) = x + 2$ dan $g(x) = x - 2$, maka fungsi $(f \cdot g)(x)$ adalah ...
  1. $x^2 + 4$
  2. $x^2 - 4$
  3. $x^2 + 4x - 4$
  4. $x^2 - 4x + 4$
  5. $2x$
3.
Diketahui $f(x) = 3x$ dan $g(x) = x + 4$. Nilai dari fungsi komposisi $(f \circ g)(x)$ adalah ...
  1. $3x + 4$
  2. $3x + 12$
  3. $3x^2 + 4$
  4. $x + 12$
  5. $3x + 7$
4.
Invers dari fungsi $f(x) = 4x - 8$ adalah $f^{-1}(x) = \dots$
  1. $4x + 8$
  2. $\frac{1}{4}x + 8$
  3. $\frac{1}{4}x + 2$
  4. $4x - 2$
  5. $\frac{1}{4}x - 2$
5.
Diketahui $f(x) = x^2 + 2x - 3$ dan $g(x) = x + 3$. Rumus fungsi $(f - g)(x)$ adalah ...
  1. $x^2 + x - 6$
  2. $x^2 + 3x$
  3. $x^2 + x$
  4. $x^2 + 3x - 6$
  5. $x^2 - x - 6$
6.
Jika $f(x) = x^2 + 1$ dan $g(x) = 2x - 1$, maka $(f \circ g)(x)$ adalah ...
  1. $4x^2 - 2x + 1$
  2. $2x^2 + 1$
  3. $4x^2 - 4x + 2$
  4. $4x^2 - 4x + 1$
  5. $2x^2 - 2x + 2$
7.
Diketahui $f(x) = 3x - 2$ dan $g(x) = x^2 + 4$. Nilai komposisi $(g \circ f)(x)$ adalah ...
  1. $9x^2 - 12x + 8$
  2. $3x^2 + 10$
  3. $9x^2 - 6x + 8$
  4. $3x^2 + 12x + 8$
  5. $9x^2 + 8$
8.
Invers dari fungsi $f(x) = \frac{2x - 3}{x + 4}, x \neq -4$ adalah ...
  1. $\frac{4x - 3}{2 - x}, x \neq 2$
  2. $\frac{-4x - 3}{x - 2}, x \neq 2$
  3. $\frac{4x + 3}{x - 2}, x \neq 2$
  4. $\frac{2x + 4}{x - 3}, x \neq 3$
  5. $\frac{x + 4}{2x - 3}, x \neq \frac{3}{2}$

13.

14.
Daerah asal (domain) dari fungsi $(f \cdot g)(x)$ jika $f(x) = \sqrt{x-2}$ dan $g(x) = \frac{1}{\sqrt{5-x}}$ adalah ...
  1. $\{x \mid 2 \leq x \leq 5, x \in \mathbb{R}\}$
  2. $\{x \mid 2 < x < 5, x \in \mathbb{R}\}$
  3. $\{x \mid 2 \leq x < 5, x \in \mathbb{R}\}$
  4. $\{x \mid x \geq 2, x \in \mathbb{R}\}$
  5. $\{x \mid x < 5, x \in \mathbb{R}\}$
15.

  2. $3 - \sqrt{x-1}$
  3. $-3 + \sqrt{x+1}$
  4. $3 + \sqrt{x+1}$
  5. $\sqrt{x-1} - 3$
16.

  2. $4x^2 + 12x + 9$
  3. $x^2 - 2x + 3$
  4. $2x^2 - 2x + 1$
  5. $4x^2 - 6x + 9$
